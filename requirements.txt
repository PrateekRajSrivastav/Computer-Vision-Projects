# Computer Vision Projects using OpenCV & TensorFlow

## 📌 Overview

This repository contains a collection of Computer Vision and Digital Image Processing projects implemented using **Python**, **OpenCV**, **TensorFlow**, **NumPy**, and **Matplotlib**. The notebook demonstrates various image processing techniques, feature extraction methods, and deep learning concepts used in modern computer vision applications.

The projects cover image enhancement, geometric transformations, histogram processing, filtering, segmentation, feature detection, edge detection, and Convolutional Neural Network (CNN)-based image classification.

---

## 🚀 Features

* Image Color Transformations (Grayscale, Binary, Sepia)
* Image Geometric Transformations (Translation, Rotation, Scaling)
* Negative, Log, and Gamma Transformations
* Histogram Equalization
* Histogram Matching
* Image Enhancement
* Spatial and Frequency Domain Filtering
* Image Restoration and Denoising
* Morphological Operations
* Pseudo Coloring
* Skin Detection using RGB and HSV Color Spaces
* Harris Corner Detection
* Boundary Linking
* Canny Edge Detection
* CNN-based Image Classification using TensorFlow (CIFAR-10)
